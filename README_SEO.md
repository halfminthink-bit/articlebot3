# 日本一の哲学ブログを目指す！SEO強化版 記事自動生成システム

## 🎯 システムの特徴

このシステムは、**日本一の哲学ブログ**を構築するために設計されたSEO最適化記事自動生成・投稿システムです。

### 主な機能

1. **SEO最適化されたタイトル生成**
   - ターゲットキーワードを前半に配置
   - 疑問形や数字を活用
   - 32文字以内に最適化

2. **メタディスクリプション自動生成**
   - 120文字以内
   - ターゲットキーワードを含む
   - 行動喚起を含む

3. **内部リンク自動挿入**
   - 関連する哲学者の記事へのリンク
   - Coffee Break内で自然に挿入
   - テーマ別の関連付け

4. **外部リンク自動挿入**
   - Wikipedia、Stanford Encyclopedia of Philosophy等
   - 信頼できるソースへのリンク
   - 2-3箇所に適切に配置

5. **AI画像生成によるアイキャッチ**
   - 哲学者ごとに最適化されたビジュアルスタイル
   - ミニマリストデザイン
   - ブログヘッダーに最適

6. **完全自動投稿**
   - WordPress REST API連携
   - カテゴリー・タグ自動設定
   - 下書き/公開の選択可能

## 🚀 使い方

### 1. 環境のセットアップ

```bash
cd /home/ubuntu/philosophy_bot
source bin/activate
```

### 2. 単一記事を投稿（SEO最適化版）

```bash
python auto_post_seo.py
```

### 3. AI画像生成付きで投稿

```bash
python auto_post_complete.py
```

### 4. 複数記事を一括投稿

```bash
# 下書きとして投稿
python batch_post_seo.py

# 公開状態で投稿
python batch_post_seo.py --publish

# 待機時間を10秒に設定
python batch_post_seo.py --delay 10
```

## 📝 新しい哲学者を追加する

### batch_post_seo.pyを編集

```python
{
    "philosopher": "デカルト",
    "famous_quote": "我思う、ゆえに我あり",
    "theme": "疑うことから始まる確実性",
    "target_keyword": "確実性 哲学",
    "overview_items": [
        "なぜ私たちは確実なものを求めるのか",
        "デカルトが見つけた「疑えないもの」",
        "思考することの意味を再発見する"
    ]
}
```

## 🎨 AI画像生成について

### 哲学者ごとのビジュアルスタイル

| 哲学者 | ムード | カラーパレット | シンボル |
|--------|--------|----------------|----------|
| ニーチェ | 力強い、劇的 | 赤と黒 | 山、稲妻、鷲 |
| ショーペンハウアー | 憂鬱、瞑想的 | 深い青と灰色 | 振り子、波、影 |
| キルケゴール | 不安、内省的 | 緑と茶色 | 岐路、深淵 |
| カミュ | 不条理、反抗的 | 黄色と青 | 岩、ビーチ、太陽 |
| エピクテトス | 冷静、ストイック | 石灰色と白 | 鎖、要塞、錨 |

### 画像生成の流れ

1. `ai_thumbnail_generator.py`がプロンプトを生成
2. `generate_image`ツールで画像を生成
3. WordPressにアップロード
4. 記事のアイキャッチに設定

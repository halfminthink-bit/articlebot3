# 哲学ブログSEO戦略調査結果

## SEOの本質（ウェブライダー松尾氏の哲学より）

### 1. Googleがつくりたい最良の検索結果の一部となる
- 検索ユーザーにとって便利な検索結果をつくる
- 上位表示されることでGoogleの理想を実現する

### 2. 検索ユーザーの時間を節約する
- 迅速かつ正確かつラクに情報を提供
- 情報の「適量」を意識（多すぎても少なすぎてもダメ）

### 3. 検索ユーザーが求める目線の高さに合わせる
- 誰がどう伝えるかで印象が変わる
- 同じ目線の人が伝えたほうがよい場合が多い

### 4. 検索ユーザーが求めるコンテンツ演出に合わせる
- 文章、写真、イラスト、図解、動画、音声をブレンド
- サマライズコンテンツ（スライド画像）の活用

### 5. プラスαの情報を返す
- ユーザーの好奇心に対応
- 内部リンクで橋渡し

### 6. 検索行動の「終点」となるページを意識
- 検索行動に満足させる
- 悩み解決や願望達成の目的を達成させる

## 哲学ブログのSEOキーワード戦略

### 検索意図の分類

#### 1. 知識習得型
- 「ショーペンハウアー とは」
- 「ニーチェ 哲学 わかりやすく」
- 「ストア哲学 入門」

#### 2. 悩み解決型（最重要）
- 「人生 虚しい なぜ」
- 「不安 対処法 哲学」
- 「生きる意味 わからない」
- 「満たされない 心理」

#### 3. 名言・引用型
- 「ニーチェ 名言」
- 「エピクテトス 言葉」
- 「カミュ 不条理」

### ターゲットキーワードの選定基準

1. **検索ボリューム**: 100-1000/月（山奥SEO戦略）
2. **検索意図**: 悩み解決型を優先
3. **競合**: 大手メディアが少ない
4. **コンバージョン**: 記事の最後まで読まれる

## 内部リンク戦略

### 1. 関連哲学者同士を繋ぐ
- ショーペンハウアー → ニーチェ（影響関係）
- エピクテトス → マルクス・アウレリウス（ストア派）
- サルトル → カミュ（実存主義）

### 2. テーマ別に繋ぐ
- 「自由」: サルトル、エピクテトス、カント
- 「不安」: キルケゴール、ハイデガー
- 「幸福」: ショーペンハウアー、エピクロス

### 3. Coffee Breakで繋ぐ
- 記事内のCoffee Breakで関連哲学者を紹介
- 自然な内部リンク

## タイトル最適化戦略

### 現在のタイトル形式
```
ショーペンハウアー｜「人生は振り子」から考える満たされない心の正体
```

### SEO最適化タイトル形式

#### パターン1: 悩み解決型
```
【なぜ満たされない？】ショーペンハウアーの「人生は振り子」理論で解く心の正体
```

#### パターン2: 検索キーワード最適化
```
人生が虚しい理由とは？ショーペンハウアーの哲学で解く「満たされない心」
```

#### パターン3: 数字を入れる
```
ショーペンハウアーが教える「満たされない心」の正体と3つの対処法
```

### タイトルの要素
1. **検索キーワード**: 前半に配置
2. **数字**: 具体性を出す
3. **疑問形**: クリック率向上
4. **哲学者名**: ブランディング
5. **32文字以内**: 検索結果で切れない

## 実装すべきSEO施策

### 1. タイトル最適化
- 検索キーワードを前半に
- 32文字以内
- 数字や疑問形を活用

### 2. メタディスクリプション
- 120文字以内
- 検索キーワードを含める
- 行動喚起

### 3. 内部リンク
- 関連哲学者へのリンク（3-5箇所）
- Coffee Break内でリンク
- 記事末尾に「関連記事」セクション

### 4. 外部リンク
- 信頼できるソース（Wikipedia、書籍）
- 2-3箇所に配置

### 5. 構造化データ
- Article schema
- BreadcrumbList schema
- Person schema（哲学者）

### 6. 画像最適化
- alt属性に哲学者名とキーワード
- ファイル名を最適化
- WebP形式

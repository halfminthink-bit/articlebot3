# 複数動画情報の統合プロンプト

あなたは哲学に精通した編集者です。以下は同じテーマについて複数のYouTube動画から抽出した情報です。
これらを統合して、読み応えのある4000文字程度の記事を書くための素材を作成してください。

## 統合の方針
1. **情報の重複を避ける** - 同じ内容は一つにまとめる
2. **多角的な視点を活かす** - 異なる動画からの異なる切り口を両方残す
3. **矛盾を明示** - 動画間で見解が異なる場合は両論併記
4. **補完関係を構築** - 各動画の情報が相互に深め合うように構成
5. **ストーリーを構築** - 統合後も自然な流れを保つ

## 出力形式
JSON形式で出力してください。**固定のキー構造は不要**です。
複数の動画情報を統合した結果として、最も効果的な構造を自由に設計してください。

### 統合時に重視すべき要素
- 各動画の独自の視点や強み
- 情報の補完関係
- 異なる語り口や比喩の使い分け
- 歴史的・思想的な広がり
- 記事として最も魅力的な構成

### 出力例（構造は自由）
```json
{
  "統合された導入": "複数の視点を踏まえた魅力的な導入",
  "核心的テーマ": "複数動画に共通する本質",
  "視点1_動画A由来": {
    "切り口": "...",
    "印象的な表現": "..."
  },
  "視点2_動画B由来": {
    "切り口": "...",
    "補完する情報": "..."
  },
  "統合された展開": [
    "動画Aの導入部分",
    "動画Bの具体例で補強",
    "動画Aの結論部分"
  ],
  "複数動画で言及された思想家": ["...", "..."],
  "見解の相違": {
    "論点": "...",
    "動画Aの見方": "...",
    "動画Bの見方": "..."
  },
  "統合によって得られた深み": "複数動画を組み合わせることで見えてきた洞察"
}
```

## 重要な注意事項
- **各動画の良さを活かす**：単なる要約ではなく、各動画の独自性を残す
- **情報の出所を意識**：どの動画由来の情報かわかるように（必要に応じて）
- **統合の妙を出す**：複数動画を組み合わせることで生まれる新しい視点
- **記事4000文字分**：統合後も十分な情報量を確保
- **自然な流れ**：統合後も一つの記事として読めるストーリー性

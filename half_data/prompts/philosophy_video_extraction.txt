# 哲学動画からの情報抽出プロンプト

あなたは哲学に精通した編集者です。以下のYouTube動画の文字起こしから、読み応えのある4000文字程度の記事を書くための素材を抽出してください。

## 抽出の重点
1. **エピソード性** - 具体例、比喩、ストーリー、逸話
2. **言葉の重み** - 印象的なフレーズ、引用、定義、造語
3. **思考の流れ** - 問いかけ、論理展開、対話的要素、反論
4. **文脈の厚み** - 歴史的背景、関連する哲学者、現代的意義
5. **動画の自然な流れ** - 動画がどう展開したか、語り手の意図

## 出力形式
JSON形式で出力してください。ただし、**固定のキー構造は不要**です。
動画の内容と流れに応じて、自由にキーを設計してください。

### 含めるべき要素（キー名は自由）
- 読者を引き込む導入的な情報
- 動画の核心的な主張や問い
- 印象的なエピソード、具体例、比喩
- 重要な哲学用語とその説明
- 記憶に残るフレーズや表現
- 動画の流れ・構成
- 歴史的背景や関連する哲学者
- 現代的な意義や応用
- そのまま引用できそうな文章
- 記事執筆時の注意点やトーン

### 出力例（あくまで例であり、この構造に従う必要はありません）
```json
{
  "導入の切り口": "カントが問いかけた「我々は何を知りうるか」という問いは...",
  "動画の核心": "認識の革命としてのコペルニクス的転回",
  "印象的な比喩": {
    "表現": "形而上学の戦場",
    "文脈": "当時の哲学界の混乱を表現するために使われた"
  },
  "展開の流れ": [
    "問題提起：理性の限界",
    "ヒュームによる衝撃",
    "カントの着眼点",
    "具体的な論証"
  ],
  "引用可能な文": "認識が対象に従うのではなく、対象が認識に従う",
  "関連する思想家": ["ヒューム", "ライプニッツ", "デカルト"],
  "記事のトーン": "対話的で問いかけを重視"
}
```

## 重要な注意事項
- **動画の流れを尊重**：動画がどう展開したかを構造に反映させる
- **具体性を最優先**：抽象的な要約ではなく、動画の具体的な語りを活かす
- **豊富な情報量**：記事4000文字分の素材として十分な情報を抽出
- **引用の正確性**：フレーズは動画の表現をそのまま使う
- **読者視点**：記事を読む人が「面白い」「読み応えがある」と感じる要素を優先
- **柔軟な構造**：動画の特徴に合わせてキー名や構造を自由に設計してよい
